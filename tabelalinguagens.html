Ranking de linguagens


Ranking Linguagens Considerações
1° C Para mim vai em primeiro, pois é a que mais domino e por ser na minha opnião a mais indicada para se começar a programar.
2° Python Uma linguagem de mais fácil aprendizado, mais versátil e mais usada em Banco de Dados, que até então tinha sido a cadeira de maior indentificação por minha parte. Muito procurada por empresas por tais motivos.
3° C# De mais fácil acesso, mais usadas por empresas pequenas/startups.
4° JavaScript Linguagem universal, voltada para front-end; Mas desenvolvedores que sabem essa linguagem são os mais procurados, depois de Python, pois são considerados praticamente full stack.
5° Java Igual a JavaScript, porém voltada para o back-end.
6° PHP Fácil e simples, mais voltada para o desenvolvimento Web, como outras, mas não tão usada como as outras.

Não tenho conhecimento sobre todas essas linguagens, então, todas as informações adicionais postas na tabela foram a partir de pesquisas, que acabaram formando uma opnião por minha parte.
